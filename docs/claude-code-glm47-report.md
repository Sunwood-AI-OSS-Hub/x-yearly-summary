## 内容分析

### テーマの共通点

すべての投稿で **Claude Code** と **GLM4.7** を組み合わせ、以下の自動化を実現しています：

| 機能 | 説明 |
|------|------|
| **Issue/PR自動処理** | GLMが作成したイシューブランチを自動でPR |
| **PRコメント対応** | PRへの追加コメントに対して自動で回答＋修正 |
| **視点制御** | ギャル化などのキャラクター設定を切り替え可能 |
| **スキル自動生成** | Claude Code の Skill を自動生成する「ZERO CC」 |

### 技術的進捗

シリーズは❶から❹へ進むにつれ、基本的な統合から高度な同期・修正機能へ発展：

1. **1月11日**: ZERO CC 開発（GLM4.7 で試験的に使用）
2. **1月12日**: GLM4.7 が Claude Code Actions で動作確認
3. **1月18日**: サーバーレスエージェント❶〜❹（集中更新）

## Claude Code で GLM4.7 を使う設定方法

### API エンドポイント

GLM4.7 は **Coding 専用エンドポイント** を提供：

```
https://open.bigmodel.cn/api/coding/paas/v4
```

### settings.json の設定

```json
{
  "api_base_url": "https://open.bigmodel.cn/api/coding/paas/v4",
  "api_key": "あなたのZhipu APIキー",
  "model": "glm-4.7"
}
```

### GLM4.7 の特徴

| ベンチマーク | スコア |
|-------------|--------|
| **SWE-bench** | 73.8%（オープンソース最高） |
| **AIME（数学）** | 95.7% |
| **HLE（複雑なツール使用）** | 42.8% |

## ZERO CC について

### 概要

「ZERO CC」は Maki が開発した **Claude Code の Skill を自動生成する Skill** です。

### 機能

- Claude Code の Skill を自動生成
- リポジトリの作成〜メンテナンスを実施
- GLM4.7 で試験的に開発中（デザイン能力を評価）

@hAru_mAki_ch の投稿は、**Claude Code x GLM4.7 の統合実験**を中心に、2026年1月中旬に活発化しています。
